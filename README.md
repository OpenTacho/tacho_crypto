# tacho_crypto

This project implements Tachograph G2 certificate verification in Rust.
Main goal is to make a no-std, no-alloc VU (Vehicle Unit) implementation on an ESP32C3 microcontroller.


Missing features:
- certificate signing
- BrainpoolP512r1
